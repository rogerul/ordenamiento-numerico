
El método que he utilizado es el algoritmo de ordenamiento Quicksort en Javascript

El algoritmo quicksort es un algoritmo de ordenamiento, como su nombre lo dice,
es de los más rápidos y es recomendable su uso (no como el de burbuja).

Se puede observar mediante una ligera animación como se va ordenando un conjunto de números por medio de este método de ordenamiento.

Una función que he  utilizado es "setTimeout", la cual se encarga de ejecutar una función o un código después de pasar cierto tiempo,
en este caso puse 500 que es medio segundo. Hago uso de esta función par que se pueda notar el ordenamiento y de un efecto de animación.
